El proyecto es lanzado desde la clase MainConcesionario.java, que se encuentra en la carpeta src/main.
Hay 3 formas de acceder al concesionario: como cliente, como trabajador o como administrador.
Para acceder como cliente puedes registrarte como un nuevo cliente, o usar como nombre de usuario marcos1234 y contraseña 1234.
Para acceder como trabajador puedes usar como nombre de usuario mikel1234 y contraseña 1234.
Para acceder como administrador puedes usar como nombre de usuario admin y contraseña admin.
